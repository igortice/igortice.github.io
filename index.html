Gem Informante

- 04/08/2015

É com muito prazer que eu disponibilizo minha primeira gem para comunidade de Ruby on Rails, Informante. Essa é uma simples gem para aplicações Ruby on Rails com o papel de centralizar dados de configurações em um único local e que possibilite o acesso a esses dados, facilmente, em qualquer ponto de uma aplicação Ruby on Rails.
